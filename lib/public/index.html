nepal-api

Use this free api to create, read, update and delete data on Nepal's caste system, districts, exports, flowers, and religions.

districts

Route Method Description
/districts GET get a list of all
/districts/:id GET get single by id
/districts POST curl -d '{"name": "< text >", "population": "< text >" }' -H 'Content-Type: application/json' 'https://nepal-api.herokuapp.com/castes'
/districts PATCH curl -d '{"id": "< text >", "name": "< text >", "population": "< text >" }' -H 'Content-Type: application/json' 'https://nepal-api.herokuapp.com/castes'
/districts DELETE curl -d '{"id": "< text >" }' -H 'Content-Type: application/json' 'https://nepal-api.herokuapp.com/castes'
